Documentación Completa de JavaScript

Un resumen de los conceptos, operadores y objetos de JavaScript vistos en los Temas 2 y 3.

1. Objeto String y Expresiones Regulares

Manipulación de Cadenas de Texto

El objeto String se utiliza para trabajar con texto. Permite realizar una gran variedad de operaciones, desde obtener su longitud hasta buscar y reemplazar subcadenas.

Creación y Propiedades:
  • Creación: Se puede crear una cadena de forma literal (let s = "hola") o como un objeto (let s = new String("hola")).
  • .length: Devuelve el número de caracteres de la cadena.
  • Caracteres de escape: Se usa la barra invertida \ para incluir caracteres especiales como comillas dentro de una cadena. Ejemplo: "El pájaro \"pedro\"...".
Métodos Fundamentales:
  • .indexOf(subcadena): Devuelve la posición (índice) de la primera aparición de una subcadena. Si no la encuentra, devuelve -1.
  • .match(expresionRegular): Busca una coincidencia entre una expresión regular y una cadena, devolviendo las coincidencias en un array.
Código de Ejemplo (Ejercicios 2 y 10):

// Creación de un objeto String y uso de propiedades/métodos
var s = new String('rafa');
document.writeln("Cadena: " + s); // Muestra 'rafa'

var longitud = s.length;
document.writeln("Longitud: " + longitud); // Muestra 4

var pos = s.indexOf("fa"); 
document.writeln("Posición de 'fa': " + pos); // Muestra 2

// Búsqueda con expresiones regulares
cadena = "Los temas son tema 1, tema 3, tema 5 y tema 6";
expresion = /(tema \d+)/i; // Busca "tema" seguido de un número
hallado = cadena.match(expresion); 
document.write("Primera coincidencia encontrada: " + hallado[0]); // Muestra "tema 1"
Explicación:
  • En el primer bloque, se crea un objeto String, se obtiene su longitud con .length y se busca la posición de "fa" con .indexOf().
  • En el segundo, se usa una expresión regular para encontrar un patrón específico ("tema " seguido de uno o más dígitos \d+) dentro de una cadena. El flag i hace que la búsqueda no distinga mayúsculas de minúsculas.

2. Objeto Array

Manejo de Colecciones Ordenadas

Los arrays son objetos similares a una lista que permiten almacenar múltiples valores en una sola variable. Los elementos se acceden mediante un índice numérico que empieza en 0.

Creación y Acceso:
  • Creación literal (preferida): let miArray = ["a", "b", "c"];
  • Creación con constructor: let miArray = new Array("a", "b", "c");
  • Acceso a elementos: Se utiliza la notación de corchetes. Ejemplo: miArray[0] devolvería "a".
Diferencia con Objetos Genéricos:

Aunque se puede crear una estructura similar a un array con un objeto ({elem1: "a", elem2: "b"}), no es un verdadero array y no tendrá acceso a sus métodos específicos (como .push(), .pop(), .forEach(), etc.). El operador instanceof Array permite verificar si una variable es un array real.

Código de Ejemplo (Ejercicios 4, 6, 8 y 12):

// Creación de un array literal
var arraya = ["Santiago", "Coruña", "Lugo"];
document.writeln(arraya[1]); // Muestra "Coruña"

// Creación de un objeto que parece un array
var arrayb = { elem1: "Santiago", elem2: "Coruña", elem3: "Lugo" };
document.writeln(arrayb["elem3"]); // Muestra "Lugo"

// El operador 'delete' elimina una propiedad de un objeto
delete arrayb.elem3;
document.writeln(arrayb["elem3"]); // Muestra 'undefined'

// Comprobar si una variable es un Array
let a = new Array(1, 2, 3);
let esArray = a instanceof Array; // Devuelve true
document.writeln("La variable 'a' es un array: " + esArray);

3. Estructuras de Control (Condicionales y Bucles)

Dirigiendo el Flujo del Programa

Las estructuras de control permiten ejecutar diferentes bloques de código según se cumplan ciertas condiciones (condicionales) o repetir un bloque de código múltiples veces (bucles).

Condicional if/else:

Ejecuta un bloque de código si la condición es verdadera, y otro bloque (opcional) si es falsa.


let num = prompt("Ingresa un número de cliente:");
if (num == 1000) { // Ojo: '==' compara valor, '===' compara valor y tipo.
    document.writeln("Ganaste un premio");
} else {
    document.writeln("Lo sentimos, sigue participando");
}
Bucle for:

Repite un bloque de código un número determinado de veces. Es ideal para iterar sobre arrays o realizar una acción N veces.


let capital = 1000;
let interesAnual = 0.05;
let numAnios = 5;

for (let i = 1; i <= numAnios; i++) {
  capital *= (1 + interesAnual);
  document.writeln("Capital tras año " + i + ": " + capital.toFixed(2) + "€<br>");
}

4. Objetos Number y Math

Manipulación de Números y Operaciones Matemáticas

Estos ejercicios muestran cómo trabajar con números, validar entradas y realizar operaciones matemáticas comunes utilizando los objetos Number y Math.

Funciones destacadas del objeto Number:
  • parseFloat(string): Convierte una cadena de texto en un número de punto flotante (decimal).
  • isNaN(valor): Comprueba si un valor "No es un Número" (Not-a-Number). Es muy útil para validar si la entrada del usuario es un número válido.
Métodos destacados del objeto Math:
  • Math.floor(x): Redondea un número hacia abajo al entero más cercano.
  • Math.ceil(x): Redondea un número hacia arriba al entero más cercano.
  • Math.round(x): Redondea un número al entero más cercano (si la parte decimal es .5 o mayor, redondea hacia arriba).
  • Math.random(): Devuelve un número pseudoaleatorio entre 0 (inclusive) y 1 (exclusive).
Código de Ejemplo (Ejercicio 8):

num = parseFloat(prompt("Ingrese un numero decimal"));
if (isNaN(num)) {
    document.writeln(`<p class="text-danger">ERROR: DEBES INTRODUCIR UN NUMERO</p>`);
} else {
    document.writeln(`Es un numero lo que has introducido ${num}.
    <ol class="list group list-group-numbered">
    <li class="list-group-item">Su floor es: ${Math.floor(num)}</li>
    <li class="list-group-item">Su ceil es: ${Math.ceil(num)}</li>
    <li class="list-group-item">Su round es: ${Math.round(num)}</li>
    </ol>`);
}

5. Objeto Date

Manejo de Fechas y Horas

El objeto Date se utiliza para trabajar con fechas y horas. Permite obtener la fecha y hora actual, así como manipular sus componentes (día, mes, año, horas, minutos, etc.).

Ejemplo de uso:

const fechaActual = new Date();
document.write("La fecha y hora actual es: " + fechaActual.toLocaleString());

// Obtener componentes individuales
let anio = fechaActual.getFullYear(); // Año con 4 dígitos
let mes = fechaActual.getMonth(); // Mes (0-11, Enero es 0)
let dia = fechaActual.getDate(); // Día del mes (1-31)

document.write(`<br>Hoy es ${dia}/${mes + 1}/${anio}`);

6. Objetos location y navigator

Ejercicios 2-11: Interactuando con el Navegador y la URL

Estos ejercicios exploran las propiedades y métodos de los objetos location (información de la URL) y navigator (información del navegador).

Funciones destacadas del objeto location:
  • Mostrar propiedades: Se accede a window.location para mostrar detalles de la URL actual como .href, .hostname, .protocol, etc.
  • location.assign(url): Navega a una nueva URL y la guarda en el historial, permitiendo usar el botón "Atrás" del navegador.
  • location.replace(url): Navega a una nueva URL pero la reemplaza en el historial. No se puede volver a la página anterior.
  • location.reload(): Recarga la página actual, equivalente a pulsar F5.
Funciones destacadas del objeto navigator:
  • Mostrar propiedades: Se accede a window.navigator para obtener información del navegador como .userAgent, .platform o si las cookies están habilitadas (.cookieEnabled).
  • Geolocalización: Se utiliza navigator.geolocation.getCurrentPosition() para solicitar al usuario su ubicación (latitud y longitud) y mostrarla en pantalla.
  • Integración con Mapas: Se manipula un <iframe> para mostrar un mapa de Google Maps, cambiando dinámicamente la URL para ajustar el nivel de zoom.

7. Objeto document

Ejercicio 1: Colecciones HTML del DOM

Este ejercicio recorre varias propiedades del objeto document que son colecciones de elementos HTML. Estas colecciones permiten acceder a todos los elementos de un tipo específico (imágenes, enlaces, formularios, etc.) presentes en la página.

Código de Ejemplo:

// Array con los nombres de las colecciones a inspeccionar.
const nombresDeColecciones = [
    "anchors", "applets", "links", "forms", "images", "scripts", "styleSheets", "cookie"
];

// Se itera sobre cada nombre.
for (let i = 0; i < nombresDeColecciones.length; i++) {
    const nombreActual = nombresDeColecciones[i];

    // Se accede a la colección dinámicamente usando document[nombreActual]
    const coleccion = document[nombreActual];

    // Se muestra el nombre de la colección y su contenido.
    document.write(`<h2>document.${nombreActual}</h2>`);

    if (coleccion && coleccion.length > 0) {
        document.write("<ul>");
        for (let j = 0; j < coleccion.length; j++) {
            // Se muestra cada elemento de la colección.
            document.write(`<li>Elemento ${j}</li>`);
        }
        document.write("</ul>");
    } else {
        document.write("<p>La colección está vacía o no es una colección iterable.</p>");
    }
}
Colecciones exploradas:

Se inspeccionan document.links, document.images, document.forms, document.scripts, document.styleSheets, y la propiedad document.cookie, mostrando cuántos elementos de cada tipo hay en la página y sus detalles.

8. Operadores en JavaScript

Realizando Comparaciones y Operaciones

JavaScript incluye un amplio conjunto de operadores para realizar tareas lógicas, matemáticas y de asignación.

Operadores Lógicos y Booleanos:

Se usan para combinar o invertir valores booleanos (true/false). Son la base de las estructuras condicionales.

  • AND (&&): Devuelve true solo si ambos operandos son verdaderos.
  • OR (||): Devuelve true si al menos uno de los operandos es verdadero.
  • NOT (!): Invierte el valor booleano (!true es false).

let a = true, b = false, c = true;
document.write("a && (b || c) = " + (a && (b || c))); // true
document.write("!(a || b) && c = " + (!(a || b) && c)); // false
Operador Ternario:

Es un atajo para una sentencia if/else simple. Su sintaxis es condicion ? valor_si_true : valor_si_false.


var a = 10, b = 15;
var opcion = b > a ? "B es mayor que A" : "A es mayor que B";
document.writeln(opcion); // Muestra "B es mayor que A"
Operador typeof:

Devuelve una cadena que indica el tipo de dato de una variable.


var a = "hola", b = 34, c = true;
document.writeln("El tipo de 'a' es: " + typeof(a)); // "string"
document.writeln("El tipo de 'b' es: " + typeof(b)); // "number"
document.writeln("El tipo de 'c' es: " + typeof(c)); // "boolean"
Operadores Bit a Bit:

Trabajan a nivel de la representación binaria de los números. Son menos comunes en el desarrollo web diario, pero muy potentes para operaciones de bajo nivel, manipulación de flags o criptografía. Incluyen AND (&), OR (|), XOR (^), y desplazamientos (<<, >>).
